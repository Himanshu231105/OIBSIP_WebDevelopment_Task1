# 🌐 Landing Page Project (Task 1)

This project is a simple and responsive **Landing Page** developed using **HTML and CSS**.  
It is created as part of the AICTE OASIS INFOBYTE Internship – Task 1.

---

## 📌 📖 Project Description

A landing page is the first page a user sees when visiting a website.  
This project focuses on building a visually appealing and well-structured webpage using basic web development technologies.

The page includes multiple sections such as a navigation bar, hero section, features section, and contact section.

---

## 🚀 ✨ Features

- Smooth scrolling navigation  
- Sticky navigation bar  
- Clean and modern design  
- Responsive layout  
- Organized sections (Home, Features, Contact)  
- User-friendly interface  

---

## 🧰 🛠️ Technologies Used

- HTML5  
- CSS3  
- Visual Studio Code  

---

## 📂 📁 Project Structure

The project contains the following files:

- index.html → Main webpage structure  
- style.css → Styling and layout  
- README.md → Project documentation  

---

## ▶️ ⚙️ How to Run the Project

Follow these steps to run the project:

1. Download or create the project folder  
2. Add all project files inside the folder  
3. Open the folder in Visual Studio Code  
4. Open the index.html file  
5. Run it using Live Server or open it in a browser  

---

## 🔗 📌 Navigation

- Home → Displays the main landing section  
- Features → Shows key highlights of the page  
- Contact → Displays contact information
